How Web pages Work

A user accesses a website by typing in a url, sending this information to a server via the Internet using HTTP. The server replies back with the data to display to the user and the browser interprets this HTTP reply and displays this information in the browser. This video is a good video to explain this.

HTML

HTML stands for Hypertext Markup Language. HTML documents form the majority of the content on the web, HTML documents contain Text, markups, images, videos, links. HTML documents contain text content which describes "what you see" and markup which describes "how it looks". This video shows this.

Tags and Elements

HTML documents are made of HTML elements. Elements are what is shown between different tags. There are also attributes which help give the property of the html tag like an image or link location. This video explains the distinction well.

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them. Block tag: will block off exactly the formatting in HTML and translate that exactly to the page in that block. Inline tags can write text all on same line with diff formatting to happen on webpage.
